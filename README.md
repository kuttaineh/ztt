# ztt
a typing tutor which speaks and gives positive feedback

Installation, from a terminal type:
```
$ git clone https://github.com/kuttaineh/ztt.git
$ cd ztt
```
MacOS & Windows ONLY:
```
$ echo "bash $HOME/ztt/zt.bash">tt
```
```
$ chmod +x tt
$ sudo mv tt /usr/local/bin
```
To run, simply type:
```
$ tt
```
